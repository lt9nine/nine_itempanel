Import Items from Lua

You can paste either a single item or multiple items in Lua format.

Example single item:

['item_name'] = {
    label = 'Item Label',
    weight = 1.0
}

Example multiple items:

{
    ['item1'] = {
        label = 'Item 1',
        weight = 1.0
    },
    ['item2'] = {
        label = 'Item 2',
        weight = 2.0
    }
}
